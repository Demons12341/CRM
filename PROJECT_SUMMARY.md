## 项目概述

我已经成功创建了一个完整的项目管理系统，采用现代化的技术栈：

- **前端**: Vue 3 + TypeScript + Element Plus + Vite

- **认证**: JWT Bearer Token

## 如何运行

### 后端运行

1. 进入后端目录
```bash
cd backend
```

2. 修改数据库连接字符串
编辑 `appsettings.json` 文件，修改 `ConnectionStrings.DefaultConnection`

3. 还原依赖包
```bash
dotnet restore
```

4. 运行后端项目
```bash
dotnet run
```

后端将运行在 `https://localhost:5000`

### 前端运行

1. 进入前端目录
```bash
cd frontend
```

2. 安装依赖
```bash
npm install
```

3. 运行前端项目
```bash
npm run dev
```

前端将运行在 `http://localhost:5173`
